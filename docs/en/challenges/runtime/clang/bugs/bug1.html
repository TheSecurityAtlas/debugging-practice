Solution: The problem is that the buffer variable does not have enough space allocated to handle what's being printed into it. This is known as a buffer overflow. If you were writing an enterprise application, hackers could easily inject code into your application and potentially take over the entire system. The solution is to increase the buffer size to anything above 41, the length of the raw message, plus 16, the maximum length of the name buffer. You also might as well add another +1 for the NULL byte.
A way we could prevent future overflows is using the "fgets" function:
// here we get rid of scanf
fgets(buffer, 64, stdin);
