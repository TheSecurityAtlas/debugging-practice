Reverse Engineering and Malware Analysis

These challenges will have you picking through binary executables for both Windows and Linux to try and determine what the file is doing and thus obtain the flag. Note that WE DO NOT HAVE REAL MALWARE SAMPLES. We are using hand written pieces of "malware" that won't actually harm your system. However, it is still recommended that you have a malware analysis VM as it is good practice.

Recommended Prerequisites

  • A solid understanding of binary formats
  • A decent understanding of assembly language
  • Knowledge of debuggers, disassemblers, and techniques for both
  • Knowledge of the computer program execution cycle

Recommended Tools

  • Disassemblers
    • Ghidra
    • Cutter
    • GNU Binary Utils - Objdump
    • IDA Pro
    • ReDASM
    • OllyDBG(Windows only)
  • Debuggers
    • GNU GDB
    • GNU GDB with the pwndbg extension
    • Radare2
    • WinDBG(Windows only)
    • OllyDBG(Windwos only)
